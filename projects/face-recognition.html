Facial Recognition System

A facial recognition pipeline built with Python and OpenCV for identity verification — the project that got me into computer vision.

Problem
Identity verification needed an approachable, from-scratch pipeline to learn how detection and classification work together on a custom dataset.
My role
Solo academic project: dataset collection and preprocessing, feature extraction, and classifier training and evaluation.
Toolkit
Python, OpenCV, Scikit-learn, Google Colab.

How it works

  • Preprocessing. Built and cleaned a custom face dataset, normalising lighting and alignment before feature extraction.
  • Feature-based classification. Used OpenCV for face detection and Scikit-learn for feature-based classification to verify identity.
  • Iteration in Colab. Trained and evaluated entirely in Google Colab, iterating quickly on preprocessing and classifier choices.

Outcome

A working identity-verification pipeline that turned classroom computer-vision theory into a concrete, testable system — and the project that first pulled me toward the computer-vision work I keep building on in later projects like the attendance kiosk.
